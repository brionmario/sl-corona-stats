## SL Corona Stats

This is a simple React app to view SL Corona stats using the API introduced by the SL Health Promotion Bureau

### Steps to run
- Clone the project

  `git clone https://github.com/vihanga-liyanage/sl-corona-stats`
  
- Run the app

  `cd sl-corona-stats`
  
  `npm run build`
  
  `npm start`
